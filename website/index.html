Email Detection
Hybrid AI engine combines a trained Logistic Regression model with rule-based pattern matching. Scans Gmail and Outlook directly, showing warnings inside the page.

File Inspection
Downloads are analyzed before execution. Extension spoofing, magic byte detection, entropy analysis, and static sandbox scanning catch disguised malware.
STATIC SANDBOX

Every detected threat is logged as forensic evidence. Filter by risk, type, or time. Auto-refreshes every 30 seconds. Full audit trail for investigation.
